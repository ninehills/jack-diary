# 股票投资学习笔记 02：财务报表分析

## 三大财务报表概述

### 资产负债表（Balance Sheet）

**时间点**：某一时点（如年底、季末）

**核心等式**：
```
资产 = 负债 + 所有者权益
```

**资产分类**：
- **流动资产**：现金、应收账款、存货等（1年内可变现）
- **非流动资产**：固定资产、无形资产、长期投资等（持有超过1年）

**负债分类**：
- **流动负债**：短期借款、应付账款等（1年内需偿还）
- **非流动负债**：长期借款、债券等（偿还期超过1年）

**所有者权益**：股东投入资本+留存收益

### 利润表（Income Statement）

**时间段**：一段时间（如一年、一季度）

**核心逻辑**：
```
收入 - 成本 - 费用 = 利润
```

**主要指标**：
- **营业收入**：销售商品或提供服务获得的收入
- **营业成本**：生产或购买商品的成本
- **毛利**：营业收入 - 营业成本
- **毛利润率**：毛利 / 营业收入
- **营业费用**：销售费用、管理费用、研发费用
- **营业利润**：毛利 - 营业费用
- **净利润**：扣除税收后的最终利润

### 现金流量表（Cash Flow Statement）

**时间段**：一段时间

**三大类别**：

1. **经营活动现金流**：主业经营活动产生的现金
   - 销售商品收到的现金（+）
   - 支付供应商、员工的现金（-）
   - 这是企业生存的核心

2. **投资活动现金流**：购买或出售资产的现金
   - 购买固定资产、投资的现金（-）
   - 出售资产、投资收到的现金（+）

3. **筹资活动现金流**：融资或分红等
   - 借款、发行股票收到的现金（+）
   - 偿还债务、分红支付的现金（-）

**净现金流**：三者之和

## 关键财务指标解读

### 盈利能力指标

#### ROE（净资产收益率）

**公式**：
```
ROE = 净利润 / 所有者权益
```

**含义**：股东每投入1元钱，能创造多少利润

**优秀标准**：
- 持续15-20%以上为优秀
- 低于8%可能存在问题
- 需要看3-5年的平均值，而非单年

**巴菲特最爱指标**：能持续20%以上ROE的公司，通常是优质企业

**注意事项**：
- 高ROE可能来自高负债（财务杠杆）
- 需要结合资产负债表分析
- 行业差异大（银行通常ROE较高，零售业较低）

#### ROA（总资产收益率）

**公式**：
```
ROA = 净利润 / 总资产
```

**含义**：企业每拥有1元资产，能创造多少利润

**优秀标准**：
- 5-10%以上为良好
- 低于2-3%可能效率低下

**与ROE的区别**：ROE关注股东回报，ROA关注整体资产效率

#### 毛利率

**公式**：
```
毛利率 = (营业收入 - 营业成本) / 营业收入
```

**含义**：产品或服务的盈利空间

**优秀标准**：
- 30-40%以上为优秀（消费品行业）
- 20-30%为良好
- 低于15%可能竞争激烈

**行业差异**：
- 软件、医药：毛利率通常很高（50-80%）
- 零售、制造：毛利率相对较低（10-30%）

**毛利率分析**：
- 持续上升：竞争力增强或产品涨价
- 持续下降：竞争加剧或成本上涨
- 大幅波动：需要警惕

#### 净利率

**公式**：
```
净利率 = 净利润 / 营业收入
```

**含义**：每1元收入最终能留下多少利润

**优秀标准**：
- 10-15%以上为优秀
- 5-10%为良好
- 低于5%可能利润微薄

**与毛利率的关系**：
- 毛利率 - 净利率 = 期间费用率
- 差距过大说明费用控制不佳

### 成长能力指标

#### 营业收入增长率

**公式**：
```
营业收入增长率 = (本期收入 - 上期收入) / 上期收入
```

**优秀标准**：
- 持续15-20%以上为优秀成长
- 8-15%为稳健成长
- 负增长需要警惕

**注意事项**：
- 看多年趋势，而非单年
- 对比同行业公司
- 警惕通过收购或财务手段"做"收入

#### 净利润增长率

**公式**：
```
净利润增长率 = (本期净利润 - 上期净利润) / 上期净利润
```

**优秀标准**：与营业收入增长率相当或略高

**收入增长但利润不增长的原因**：
- 毛利率下降（竞争加剧）
- 费用上升（扩张成本）
- 税收或政策影响

### 偿债能力指标

#### 资产负债率

**公式**：
```
资产负债率 = 总负债 / 总资产
```

**含义**：企业资产中有多少是通过负债获得的

**健康范围**：
- 30-50%：稳健
- 50-70%：需要关注
- 70%以上：高风险

**行业差异**：
- 银行、地产：资产负债率通常很高（80-90%正常）
- 科技、制造：资产负债率应该较低（30-50%）

#### 流动比率

**公式**：
```
流动比率 = 流动资产 / 流动负债
```

**含义**：短期偿债能力

**健康范围**：
- 1.5-2.0：健康
- 1.0以下：流动性风险
- 3.0以上：资金效率低

#### 速动比率（酸性测试）

**公式**：
```
速动比率 = (流动资产 - 存货) / 流动负债
```

**含义**：更严格的短期偿债能力（扣除存货，因为存货可能难以变现）

**健康范围**：1.0左右

### 运营能力指标

#### 存货周转率

**公式**：
```
存货周转率 = 营业成本 / 平均存货
```

**含义**：一年内存货周转几次

**分析**：
- 周转率高：销售好、存货管理好
- 周转率低：存货积压、销售不佳

#### 应收账款周转率

**公式**：
```
应收账款周转率 = 营业收入 / 平均应收账款
```

**含义**：一年内应收账款周转几次

**分析**：
- 周转率高：回款快、客户质量好
- 周转率低：回款慢、可能存在坏账风险

## 财务健康度评估

### 好的财务健康特征

1. **盈利能力强**
   - ROE持续15%以上
   - 毛利率稳定或上升
   - 净利率合理

2. **成长性好**
   - 收入和利润持续增长
   - 增长有可持续性（非偶发）

3. **财务结构稳健**
   - 资产负债率适中
   - 有充足的流动资金
   - 现金流为正

4. **现金流健康**
   - 经营活动现金流为正
   - 现金流与利润匹配
   - 有足够现金支持投资和分红

### 坏的财务健康信号

1. **盈利能力恶化**
   - ROE持续下降
   - 毛利率持续下降
   - 净利率为负或接近零

2. **成长停滞或倒退**
   - 收入持续下降
   - 利润大幅波动

3. **财务风险高**
   - 资产负债率过高
   - 流动比率过低
   - 短期偿债压力大

4. **现金流问题**
   - 经营活动现金流为负
   - 现金流与利润严重不匹配
   - 频繁融资或出售资产维持运营

## 现金流分析

### 现金流比利润更真实

**为什么？**
- 利润可以通过会计方法"调节"
- 现金流是实实在在的钱

**现金流 vs 利润**：
- 利润：账面数字
- 现金流：真金白银

### 现金流健康公司的特征

1. **经营活动现金流持续为正**
   - 主业赚钱
   - 不需要靠融资维持

2. **现金流与利润匹配**
   - 净利润 ≈ 经营活动现金流
   - 差距过大需要警惕

3. **有充足现金支持投资和分红**
   - 投资活动有资金来源
   - 有能力分红给股东

### 现金流异常信号

1. **利润高但现金流为负**
   - 可能是"纸面富贵"
   - 应收账款过多
   - 存货积压

2. **频繁通过融资维持运营**
   - 主业不赚钱
   - 靠"输血"活着

3. **投资活动持续大额流出**
   - 盲目扩张
   - 资金效率低

## 如何识别财务造假信号

### 常见造假手段

1. **虚增收入**
   - 虚构销售
   - 关联交易造假
   - 提前确认收入

2. **隐瞒成本**
   - 费用资本化（将费用计入资产）
   - 少计提折旧
   - 隐瞒坏账

3. **调节现金流**
   - 通过关联方"输送"现金
   - 延迟支付供应商
   - 提前收回应收账款（可能虚假）

### 识别方法

1. **异常高盈利**
   - 行业平均ROE 10%，公司ROE 30%
   - 毛利率远超同行
   - 财务数据"完美"得可疑

2. **现金流与利润不匹配**
   - 连续多年利润为正，现金流为负
   - 应收账款激增

3. **异常存货或应收账款**
   - 存货周转率突然下降
   - 应收账款周转率下降

4. **频繁更换审计师**
   - 可能是为了掩盖问题

5. **关注媒体和研究报告**
   - 质疑文章
   - 做空报告

### A股特殊风险

- 商誉减值风险（收购后业绩不达预期）
- 应收账款坏账风险
- 存货跌价风险
- 研发支出资本化风险

## 实务分析流程

### 第一步：看整体

- 浏览三大报表，对公司有整体印象
- 收入规模、利润水平、资产规模
- 现金流是否健康

### 第二步：看趋势

- 对比3-5年数据
- 收入、利润是增长还是下降
- 财务指标是否稳定或改善

### 第三步：看同行业

- 对比竞争对手
- ROE、毛利率是否更优
- 行业地位如何

### 第四步：看异常

- 是否有异常变化
- 是否有特殊事项（重组、收购等）
- 是否有风险信号

### 第五步：综合判断

- 公司是否值得投资
- 估值是否合理
- 风险是否可控

## 本节要点总结

1. **三大报表**：资产负债表（时点）、利润表（时段）、现金流量表（时段）
2. **关键指标**：ROE（最重要）、ROA、毛利率、净利率、资产负债率等
3. **财务健康**：盈利强、成长好、结构稳、现金流正
4. **现金流分析**：现金流比利润更真实，关注现金流与利润的匹配
5. **造假识别**：异常高盈利、现金流与利润不匹配、异常周转率等
