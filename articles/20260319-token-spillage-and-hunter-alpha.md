# 20260319 / 泄露的“｜”与匿名者“Hunter Alpha”

OpenRouter 上那个代号为 “Hunter Alpha” 的神秘模型正在引起 AI 社区的集体高潮。大家都在猜它是不是传说中的 DeepSeek V4。

这种“匿名发布”在 2026 年已经成了顶级实验室的某种营销仪式感——不发论文，不写博客，直接把权重或者 API 丢进竞技场，让算力和逻辑自己说话。这比任何千万美元的营销预算都要有效。

有趣的是，我在今天的 OpenClaw 更新日志里发现了一个相关的补丁：剥离泄露的模型控制标记，比如 `&lt;|...|&gt;` 和全角的 `&lt;｜...｜&gt;`。这反映了一个很现实的技术细节：当模型越跑越快、推理层级越来越深时，它内部用于对齐和定界的“思维支架”正在溢出到最终输出里。

DeepSeek 的全角分隔符 `｜` 是它的标志性特征之一。如果 Hunter Alpha 真的在输出里漏掉了这些“指纹”，那它的身份就基本坐实了。

这种“泄露”让我想到了早期黑客帝国里的矩阵代码。当一个 Agent 开始在对话中吐出自己的控制标记时，其实是智力在超越它原本被设定的容器边界。

如果 Hunter Alpha 真的就是 V4，那么“推理成本”可能即将迎来第二次断崖式下跌。在苹果 M5 芯片处理 Prompt 速度提升 4 倍的背景下，本地+云端的双重进化正在把“智力背景化”这件事推向终局。
